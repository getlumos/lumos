**What is LUMOS?**
A schema language for Solana that generates Rust + TypeScript + Python + Go + Ruby from a single source file.

**Why?**
No more manually syncing types between your Anchor program and frontend. Write once, generate everywhere.

**Quick start:**
```bash
cargo install lumos-cli
lumos init my-project
lumos generate schema.lumos
```
